# 🌍 Staysphere

Staysphere is a full-stack travel accommodation web application inspired by Airbnb.  
Users can explore stays, add new listings, edit properties, and leave reviews.

---

## 🚀 Features

- 🏠 Add, Edit and Delete Listings
- ⭐ Add Reviews with Ratings
- 🖼️ Image Upload via URL
- 🔐 User Authentication (Login / Register)
- 📍 Location and Pricing Details
- 🛡️ Server-side Validation
- 💾 MongoDB Database Integration
